Rippleeffect.js Documentation
What is Rippleeffect.js?
Rippleeffect.js is a lightweight JavaScript library that adds smooth ripple effects to your clickable elements. It works on any HTML element and is fully customizable.
How to Use
In HTML:
1. Include the script in your HTML:
<script src="rippleeffect.js"></script>
2. Add ripple-black or ripple-white class to any clickable div.
3. Style your divs with CSS as needed.

In React:
1. Import the script in your component or index.html:
import '../path/to/rippleeffect.js';
2. Add the classes to any div you want ripple effects on:
<div className="greenbtn ripple-black">Click Me</div>
3. Ensure your CSS for buttons is imported.
4. Rippleeffect.js will automatically attach ripple effects to divs with the classes.
